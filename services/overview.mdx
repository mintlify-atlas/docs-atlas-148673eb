## Introduction

ScaleTail provides a comprehensive catalog of **95+ pre-configured services** with Tailscale sidecar integration. Each service includes ready-to-use Docker Compose configurations that seamlessly integrate Tailscale's secure networking capabilities, making it easy to self-host applications with zero-config networking.

## What's Included

Each service configuration includes:

- **Docker Compose file** with Tailscale sidecar
- **Environment variables** and configuration options
- **Setup instructions** specific to the service
- **Network routing** via Tailscale Serve or Funnel
- **Authentication options** using Tailscale auth keys

## Getting Started

1. Browse the category pages to find services that fit your needs
2. Visit the GitHub repo for detailed setup instructions
3. Copy the Docker Compose configuration
4. Add your Tailscale auth key
5. Deploy with `docker compose up -d`
